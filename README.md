# robots-harmony
Robots Harmony

Το Robots Harmony είναι ένα μικρό frontend project φτιαγμένο με HTML, CSS και καθαρή JavaScript.
Η ιδέα ήταν να δουλέψουμε πάνω στη δομή, στο DOM και στην αλληλεπίδραση. Ό,τι φαίνεται στην οθόνη αντιστοιχεί άμεσα σε κώδικα.

Λειτουργία:

Η σελίδα εμφανίζει μια γκαλερί αποτελούμενη από 4 ρομπότ.
Κάθε ρομπότ αποτελείται από απλά HTML στοιχεία (μάτια, χέρια, πόδια, κορμό) και μπορεί να αλλάζει χρώματα μέσω πληκτρολογίου.
Υπάρχει επίσης ήχος στο hover, ενεργός μόνο αφού ο χρήστης κάνει ένα αρχικό click, λόγω των περιορισμών των browsers.

Πώς είναι στημένο:

Το project είναι χωρισμένο:
	•	HTML για τη δομή
	•	CSS για layout, χρώματα και animations
	•	JavaScript για events και αλλαγές κατάστασης

Δεν χρησιμοποιείται καμία εξωτερική βιβλιοθήκη.

Αλληλεπίδραση:
Κάθε ρομπότ αντιδρά σε συγκεκριμένα πλήκτρα του πληκτρολογίου, τα οποία αλλάζουν χρώμα σε διαφορετικά μέρη του σώματός του. Η λογική είναι απλή και φαίνεται άμεσα μέσα στο αρχείο JavaScript. 
Εκτέλεση:
Απλώς ανοίγουμε το index.html σε έναν σύγχρονο browser. Δεν χρειάζεται εγκατάσταση ή setup.
